PIXELATEDVOLUME'S LOTC CUSTOM ITEMS
===================================

This is a Minecraft resource pack for custom items used on the 
Lord of the Craft (LotC), a medieval-fantasy roleplay server.
It requires the InexactCIT mod to work correctly, which can be
found at modrinth.com/mod/inexactcit.

Be aware all larger-than-standard models render too large in the 
inventory GUI.  A suitable solution will be implemented as soon 
as it is discovered.

This pack was created and is maintained by PixelatedVolume, who 
may be contacted at pixelatedvolume on Discord.

The artist of every texture is credited in the item model file
corresponding to the texture.
